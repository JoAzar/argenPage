    Entiendase [012 345 67 89 xx] como los componentes de la interfaz de red

    012

    indica que es una interfaz inalámbrica

    345

    slot del bus PCI y la posición de la interfaz inalámbrica en el sistema

    67

    interfaz Ethernet

    89

    Corresponde al slot del bus PCI donde está ubicada la tarjeta de red

    xx

    Indica la interfaz número 0 en ese slot PCI. Si hubiera más de una interfaz en el mismo slot, se usarían s1, s2, y así sucesivamente

    UDP (User Datagram Protocol)

    Es un protocolo de capa de transporte en el modelo OSI que No incluye mecanismos de control de flujo, control de congestión ni retransmisión de datos perdidos, lo que significa que los datos pueden llegar desordenados, duplicados o incluso perderse sin que el protocolo lo detecte.

    TCP (Transmission Control Protocol)

    Es un protocolo de capa de transporte en el modelo OSI que proporciona una comunicación confiable y orientada a la conexión entre aplicaciones en redes IP

    Address Resolution Protocol

    Resuelve las direcciones IP a direcciones MAC dentro de una local network

    ARP On

    Herramienta de seguridad para sistemas Linux que se utiliza para detectar y prevenir ataques relacionados con la manipulación de direcciones ARP (Address Resolution Protocol)

    sudo apt install arpon

    Instalación de la herramienta (comando de Linux en general)

    sudo arpon -i interfaz -S

    Escaneo de interfaz de red en tiempo real

    Métodos de detección

    1 - SARPI (Static ARP Inspection) for the statically configured networks without DHCP

    2 - DARPI (Dynamic ARP Inspection) for the dynamically configured networks with DHCP

    3 - HARPI (Hybrid ARP Inspection) for the statically and dynamically configured networks with DHCP

    -H para HARPI | -S para SARPI | -D para Darpi
